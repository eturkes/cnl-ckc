- SWI stream `encoding(utf8)` is not strict octet validation; deterministic adapters must read bytes and apply explicit UTF-8 boundary, overlong, surrogate, and maximum-code-point checks before parsing.
- `numbervars/3` collides with pre-existing `'$VAR'/1` terms; validate acyclicity, attributed variables, atomic kinds, and reserved `'$VAR'/1` subterms before numbering, including error terms so fallback bytes stay deterministic.
- APE noun copula fixtures require the indefinite article (`John is a patient.`); bare `John is patient.` fails ACE parsing, and the admitted noun anchor is token 4.
- APE DRS negation operators are distinguished by arity and spelling: `-/1` = classical ¬ ("does not"/"no"/"it is false that"), `~/1` = NAF ("... not provably ..." VP form, "It is not provable that ..."), `-/2` = the sentence/token anchor wrapper (never negation). Neither negation operator carries its own anchor; anchors stay on the inner conditions, and an antecedent-bound referent appears free inside the negated sub-DRS.
- Program digests hash the original pre-decode byte list, including the final LF, via `crypto_data_hash/3` with `encoding(octet)`; the default hashes characters.
- Wh answer order = canonical-serialization bytes of the ground atom; quoting participates (`'z z'` 0x27 sorts before `a` 0x61), never insertion or standard term order.
- Answer digests are output-only: no digest line appears in a `run` input, so tamper gates mutate program bytes and independently recompute SHA-256.
- Prolog profile validators must not match admitted constants by unifying input terms: decompose first and compare with `==`/numeric checks. Canonical native variables can otherwise be silently instantiated, including aliases to already-validated referents.
- Vertical-slice gates must feed each stage the fresh prior-stage bytes, compare them with that stage's golden before advancing, and define zero-write as no non-empty artifact: shell redirection pre-creates the failing stage's empty output file.
- Special-case canonical serializers must prove the complete well-formed shape before committing to their branch; malformed terms fall through to the generic serializer so semantic passes retain ownership of documented error classes.
- Join evaluation must fuse solution enumeration with growing-store deduplication; materializing all assignments before dedup makes independent body-only variables consume `n^k` candidate memory even when they derive one ground head.
- Canonical hex-digest atoms flip quoting by first character: digit-leading digests must be quoted (`'08d4…'`), letter-leading digests must be bare (`d28d…`) — writing a letter-leading digest quoted fails the canonical fixed point (`canonical` codepoint offset error).
- Strict UTF-8 harness cases should generate invalid octets only in scratch; for a valid multibyte byte-round-trip, use an atom that must stay quoted (docids are ASCII-only), validate it, then reserialize the validated terms because the `validate` CLI intentionally emits no stdout.
- APE Ulex proper-name lookup is case-sensitive to the ACE surface: capitalized hyphenated names work when `pn_sg/3` uses the exact capitalized WordForm (and stable exact-case lemma); a lowercased WordForm produces undefined-word warnings.
