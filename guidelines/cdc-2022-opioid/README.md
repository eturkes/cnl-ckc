# CDC 2022 opioid guideline source

This directory contains source evidence and project-authored formal derivatives for the
following publication:

Dowell D, Ragan KR, Jones CM, Baldwin GT, Chou R. CDC Clinical Practice Guideline for
Prescribing Opioids for Pain — United States, 2022. MMWR Recomm Rep
2022;71(No. RR-3):1–95. DOI `http://dx.doi.org/10.15585/mmwr.rr7103a1`.

## Repository materials

The byte-authoritative acquired PDF and the verbatim Box 3 extraction evidence are under
`guidelines/cdc-2022-opioid/source/`. The extraction preserves CDC statement text; its
number labels are presentation scaffolding outside the registered recommendation byte
ranges.

ACE authoring, terminology choices, mappings, logical projections, and other formal
representations in this directory are project-authored derivatives. They are not altered
CDC text and must not be presented as CDC-authored formalizations.

The normative source, rights, extraction, and region metadata is in
`guidelines/registry.pl`; its format is defined by `docs/registry.md`.
